Toute l'application fonctionne via du clic button
Vous avez deux écrans :
	L'écran d'accueil (faute d'assets j'ai réutilisé celle du défi) avec deux bouttons :
		Play permettant de lancer l'application
		Exit permettant de la cliquer

	L'écran principal composé de :
	Au centre une map déjà chargé
	En haut à roite de l'écran, le tileset, cliquer sur la tile de votre choix puis sur la tile que vous souhaitez 	remplacer
	En haut à gauche deux bouttons, un bouton save sauvegardant la map dans un fichier xml
	Un boutton importer qui ne fonctionne pas

A n'importe quel moment vous pouvez appuyez sur échap pour quitter l'application
